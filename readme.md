Contao Maps (MapLink)
======================

Requirements:
----------------------
    * Contao  3.3+
    * PHP     5.4+
    * jQuery  1.11.0

Description
----------------------
English:
    Contao Maps (MapLink) is a Contao extension for device-dependent delivery of cards Materiel with referencing to the native application.
